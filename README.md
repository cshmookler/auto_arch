# auto_moos

Automatically installs [MOOS](https://github.com/cshmookler/moos) on a device.

## Usage

Open a shell in an existing MOOS installation (or live ISO).

As root, execute the "auto_moos.py" script with Python.

```bash
python auto_moos.py
```
